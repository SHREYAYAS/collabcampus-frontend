# CollabCampus Frontend

React + Vite + MUI dashboard for team collaboration and project tracking.

## Scripts

- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build production: `npm run build`

## Notes

- Requires Node 18+.
- API base URL is configured in `src/api/client.js` and can be overridden via environment variables (see `.env.example`).
